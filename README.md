# 남구 · 대구광역시

> 남구 지역 정보 랜딩 사이트

## 🌐 랜딩 페이지

### **https://kr-daegu-nam.github.io/**

남구는 대구의 상징인 앞산이 병풍처럼 남쪽을 감싸고, 신천이 동쪽 경계를 따라 흐르는 도심 속 주거·문화 자치구입니다. 안지랑 곱창골목과 앞산카페거리로 대표되는 서민형 먹거리 문화, 대명동 일대의 공연예술 생태계, 이천동 고미술거리의 옛 정취가 어우러져 '남다른 남구'라는 별칭을 얻었습니다. 임진왜란 때 귀화한 명나라 장수 두사충이 고국을 그리며 이름 붙였다는 대명동(大明洞) 지명 유래는 전국 어디에도 없는 남구만의 이야기입니다. 대구교육대학교와 영남이공대학교가 자리 잡아 젊은 층 유동이 활발하고, 앞산 케이블카와 전망대는 대구 야경 명소로 이름이 높습니다. 면적은 17.5㎢로 대구 자치구 중에서도 아담하지만, 산과 하천과 문화가 촘촘히 밀집한 것이 남구의 매력입니다.
